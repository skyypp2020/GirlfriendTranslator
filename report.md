# Abstract

本專案旨在開發一款名為「女友翻譯機（Girlfriend Translator）」的互動式應用程式，解決現代情侶間常見的溝通隔閡與語意誤解問題。透過整合 Streamlit 網頁框架與以 `aisuite` 為基礎的多模組大型語言模型（LLM）介面，本系統能即時分析使用者的輸入語句，並扮演「求生型男友」的角色進行深度語意解析。

系統核心功能利用生成式 AI 的自然語言理解能力，將女友看似平常的話語轉譯為四個維度：官方說法（表面字義）、內心 OS（深層情緒）、真正意思（實際訴求），以及最關鍵的「男友存活率最高回覆」。在技術實作上，本專案展示了如何利用 Python 快速建構輕量級 Web App，並透過環境變數安全管理 API 金鑰，實現了對 Groq、OpenAI 及 Mistral 等不同 AI 供應商模型的靈活切換。

本應用程式不僅是一個技術展示，更嘗試以幽默且寫實的方式，利用人工智慧輔助情感溝通，降低對話中的摩擦成本。透過直觀的介面與即時的反饋，使用者能更準確地掌握對話情境，從而做出最適當的回應，展現了 AI 在人際關係輔助應用上的潛力與趣味性。
